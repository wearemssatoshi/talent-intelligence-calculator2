# Momentum Peaks 拠点定指数解説

## 概要
SATが開発した独自の需要予測指標「Momentum Peaks（モメンタムピークス）」の拠点別定指数。

## 対象拠点
1. **藻岩山**（もいわやま）- The Jewels
2. **大倉山**（おおくらやま）- ヌーベルプース大倉山
3. **テレビ塔** - ザ ガーデン サッポロ

## 指数体系（④TOTAL拠点指数の構成）

### ①月別季節指数 (Scale: 1-5)

| 月 | 指数 | ポジティブ要因 | ネガティブ要因 |

| 2月 | 3 | 雪まつり | 冬の出控え |
| 3月 | 3 | 春・雪解け | - |
| 4月 | 1 | 春・GW | 運休(藻岩山) |
| 5月 | 3 | 春・GW・もいわ山の日 | GW明け反動 |
| 6月 | 4 | 初夏・新緑・よさこい・神宮祭 | - |
| 7月 | 5 | 夏・ビアガーデン・PMF・花火 | - |
| 8月 | 5 | 夏休み・北海道マラソン | - |
| 9月 | 5 | オータムフェスト | - |
| 10月 | 5 | 秋・紅葉 | - |
| 11月 | 3 | ホワイトイルミネーション | 端境期 |
| 12月 | 5 | クリスマス・イルミネーション | 冬の出控え |

### ②月別曜日指数 (Scale: 1-5)
曜日ごとの需要パターン

| 曜日 | 基本指数 | 備考 |
|-----|---------|------|
| 日 | 4 | 週末需要 |
| 月 | 2 | 週明け閑散 |
| 火 | 2 | 平日 |
| 水 | 2 | 平日 |
| 木 | 3 | 週末前 |
| 金 | 4 | 週末需要開始 |
| 土 | 5 | 最大需要日 |
| 祝休 | 4 | 祝日・連休 |
| 連休中日 | 3-5 | 状況による |

---

### ③月別来場者指数 (Scale: 1-5)
実際の来場者数データに基づく指数

**計算式**:
```
来場者指数 = 実来場者数 ÷ 基準来場者数 × スケール調整
```

**データソース**:
- 藻岩山: ロープウェイ+ミニケーブル乗車人数
- 大倉山: ジャンプ場入場者数
- テレビ塔: 展望台入場者数
- 赤れんが: (理論整理中)

---

### ④TOTAL拠点指数
**計算式**:
```
TOTAL拠点指数 = (①季節指数 + ②曜日指数 + ③来場者指数) / 3
```

または加重平均:
```
TOTAL = ①×α + ②×β + ③×γ  (α+β+γ=1)
```
