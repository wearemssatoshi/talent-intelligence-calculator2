# Momentum Peaks Calculator

モメンタムピークス（需要係数）を計算するスキル。

## 概念

**モメンタムピークス** = 観光地の忙しさを数値化した独自指標

### 計算要素
1. **曜日係数**: 土日祝 > 金曜 > 平日
2. **季節係数**: 繁忙期（GW、お盆、年末年始）
3. **イベント係数**: 近隣イベントの規模
4. **天気係数**: 晴れ > 曇り > 雨

### 出力
- **0-30**: 低需要（スタッフ最小構成）
- **31-60**: 通常（標準構成）
- **61-85**: 高需要（増員推奨）
- **86-100**: ピーク（フル稼働）

## 使用方法

```bash
python .agent/scripts/momentum_calculator.py <売上CSV>
```

## 必要データ
- 過去1年分の日別売上データ
- 日付、売上、客数の列

## 出力
- 各日のモメンタム値（0-100）
- 曜日別・月別の平均モメンタム
- ピーク日の特定
