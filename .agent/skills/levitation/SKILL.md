# 🎈 Levitation Skills（浮揚スキル）

業務の「重力」を消し、現場を軽くするスキル群。

## 含まれるスキル

### excel-reader/
Excel/CSVデータを読み込む基盤スキル。

```bash
python .agent/scripts/excel_parser.py <ファイルパス>
```

### shift-solver/ (予定)
モメンタム×タレントインテリジェンスで最適シフトを計算。

### inventory-calc/ (予定)
在庫・発注量の自動計算。

## 目的
店長が**シフト作成や在庫管理**に費やす時間をゼロにし、接客に100%集中できる環境を作る。
