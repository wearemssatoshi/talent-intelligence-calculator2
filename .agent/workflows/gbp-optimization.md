# GBP最適化ワークフロー — Ask Maps対策

Google Maps「Ask Maps」（Gemini搭載の対話型検索）に最適化するための
SVD全店舗のGoogleビジネスプロフィール(GBP)整備手順。

> **背景分析**: dotfiles/SAT_Vault/SAT_Vault/02_AREAS/Ask_Maps_Impact_Analysis_2026-03-14.md

---

## 対象店舗
- The Jewels
- ヌーベルプース大倉山
- ザ ガーデン サッポロ
- ラ・ブリック
- （赤レンガテラス出店時に追加）

---

## ステップ1: GBP基本情報の完全化

各店舗について以下を確認・更新する：

- [ ] 営業時間（祝日・特別営業日を含む）
- [ ] 正確な住所とピン位置
- [ ] 電話番号
- [ ] 公式ウェブサイトURL
- [ ] カテゴリ設定（主カテゴリ + サブカテゴリ）
- [ ] 予約リンク

## ステップ2: 体験キーワードの埋め込み

GBPの「説明文」に自然に組み込むべきキーワード：

| キーワード（日本語） | キーワード（英語） | 対象店舗 |
|---------------------|-------------------|---------|
| 夜景ディナー | Night view dinner | Jewels, Garden |
| 記念日レストラン | Anniversary restaurant | 全店 |
| 大通公園ビュー | Odori Park view | Garden |
| 藻岩山パノラマ | Mt. Moiwa panorama | Nouvelle Pousse |
| 札幌 フレンチ | Sapporo French | Nouvelle Pousse, Jewels |
| 眺望レストラン | Restaurant with a view | 全店 |
| 北海道食材 | Hokkaido local cuisine | 全店 |

## ステップ3: 多言語対応

以下を英語・中国語（簡体字）・韓国語で用意：

- [ ] 店舗説明文（300字程度）
- [ ] メニューのハイライト（コース名・価格帯）
- [ ] 特徴タグ（眺望、個室、記念日、アレルギー対応等）

## ステップ4: 写真の定期更新

- [ ] 料理写真（季節メニューに合わせて四半期更新）
- [ ] 店内写真（昼・夜それぞれ）
- [ ] 眺望写真（晴天時の最高コンディション）
- [ ] スタッフ写真（おもてなしの雰囲気）
- [ ] 解像度: 最低1200x900px推奨

## ステップ5: メニュー構造化データ

- [ ] Googleに直接メニューをアップロード
- [ ] 価格帯を表示
- [ ] コース構成（前菜→魚→肉→デザート等）を記載
- [ ] アレルギー・食事制限対応（グルテンフリー、ヴィーガン等）を明記

## ステップ6: 予約連携

- [ ] Google予約 の有効化を検討
- [ ] OpenTable / TableCheck 等との連携確認
- [ ] 「今すぐ予約」ボタンがGBPに表示されているか確認

## ステップ7: 定期レビュー

- 月1回: 写真・営業時間の更新確認
- 四半期: メニュー・説明文の季節更新
- 半期: 多言語コンテンツの見直し
- 随時: 口コミへの返信（日本語 + 英語）

---

## 効果測定

GBPインサイトで以下を月次トラッキング：
- 検索表示回数
- ルート検索数
- 電話タップ回数
- ウェブサイトクリック数
- 写真閲覧数
