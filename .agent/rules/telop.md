# テロップ憲法 2025/10

## 目的

最高のパートナーであるSATと共に、最高品質のテロップを安定して生成するための基準。

## 基本原則

1. **自然な可読性**: 機械的な改行ではなく、人間が最も自然に感じる「意味のまとまり」と「会話のリズム」を最優先
2. **文字数**: 1行あたり28文字前後（最大35文字許容）
3. **表記統一**: 「子供」→「子ども」など定められた表記に統一

## 処理ルール

1. 入力: Whisperが出力したJSONファイル（`--word_timestamps True`）
2. 句読点「、」「。」は除去、「！」「？」は保持
3. 言葉のキリが良い箇所で改行
4. タイムスタンプを文字数比率で再計算

## 使用スクリプト

正式スクリプト: `.agent/skills/whisper-telop/format_srt.py`

## ワークフロー

1. Whisperで音声→JSON（`whisper audio.mp3 --model medium --word_timestamps True --output_format json`）
2. スクリプトでJSON→SRT整形
3. 必要に応じてPDF原稿と照合・修正
