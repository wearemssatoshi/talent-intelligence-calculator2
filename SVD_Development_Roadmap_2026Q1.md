## 📋 プロジェクト詳細

### ① Momentum Peaks（繁閑数値化）
**Status**: 開発中

| 項目 | 内容 |
|------|------|
| 目的 | 拠点別365日の繁閑を数値化 |
| データ | 週別、月別、拠点別の過去データ |
| 出力 | 繁閑スコア、予測客数、必要人員数 |

### ② Talent Intelligence（スキル数値化）
**Status**: ブラッシュアップ予定

| 項目 | 内容 |
|------|------|
| 目的 | スタッフの能力とスキルを数値化 |
| 要素 | スキル、経験、責務、役割 |
| 出力 | スタッフ戦力値、成長曲線 |

### ③ シフト自動作成アプリ
**Status**: 設計段階

| 項目 | 内容 |
|------|------|
| 算出式 | Momentum Peaks × Talent Intelligence |
| 入力 | 繁閑データ、スタッフスキル、希望シフト |
| 出力 | 最適化されたシフト表 |

### ④ ワインリスト 1STOPアプリ
**Status**: 要件定義

| 項目 | 内容 |
|------|------|
| 機能 | ワインリスト、棚卸し表、コントローラー |
| 目的 | 飲料管理の一元化 |

### ⑤ 全自動PL作成アプリ
**Status**: 構想段階

| 項目 | 内容 |
|------|------|
| 入力 | 売上、仕入、人件費、経費 |
| 出力 | 損益計算書（自動生成） |

### ⑥ バーチャル予約システム
**Status**: Genie連携待ち

| 項目 | 内容 |
|------|------|
| 連携 | TableCheck |
| 体験 | 3D空間で席を選んで予約 |
| 技術 | Genie（Google AI）|
