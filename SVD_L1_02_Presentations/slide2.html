Logical Tasting Methodology

3. 香りの分析 (Nose)

  • プライマリー: 品種・発酵 (果実、花)
  • セカンダリー: 醸造 (MLF、酵母、樽)
  • ターシャリー: 熟成 (瓶内、酸化)

4. 味わいの分析 (Palate)

  • 第1幕(アタック): 第一印象
  • 第2幕(ミドル): 中心の展開
  • 第3幕(アフター): 強度(パワー)
  • 質(Quality): 量より質。酸とタンニン

5. 結論・総合評価 (Conclusion)

  • 推論: マクロ(気候) → ミクロ(畑)
  • 評価: 品質、熟成、飲み頃
  • 提案: 温度、グラス、料理

Logical Tasting Goal

ワインの声を聴き、
その物語を正確に翻訳する。
