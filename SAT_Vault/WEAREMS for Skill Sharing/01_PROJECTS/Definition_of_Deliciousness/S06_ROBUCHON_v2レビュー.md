## 2. 「あと一歩」の磨きポイント — 他の5体が見逃しそうな細部

### 🔍 指摘①: v2.0方程式のスコアレンジが未定義

**問題**: Core = (FD① + FD②) × (SV③ + SV④) の理論値レンジが本文中に明示されていない。

```
理論最小値: (0+0) × (0+0) = 0
理論最大値: (10+10) × (10+10) = 400
```

しかしFD①は5サブ項目の平均（五味・風味・食感・供出温度・技術的完成度）、FD②は3サブ項目（第一印象・食材の質・独自性）。**サブ項目数が異なるのに同じ10点満点に丸めている理由**が書かれていない。

→ **修正案**: 「各カテゴリは各サブ項目の算術平均とし、10点満点に正規化する」と一文追記すべき。

### 🔍 指摘②: ルーブリックにFD①「火入れの精度」が独立項目として存在しない

「技術的完成度」に含まれるとも読めるが、世界のファインダイニング評価において**火入れ（cuisson）** は独立軸。「技術的完成度」の例示に「火入れ」が一言出るが、ルーブリック表では「火入れ、ソース、発酵等」と並列扱い。

→ **対応案**: 現行の5サブ項目を維持するなら、「技術的完成度」の判断基準に「※本項目は火入れの精度を中心に評価する」と注釈を追加。

### 🔍 指摘③: Place係数テーブルの「標準的都市型 = 1.00」問題

Place係数が1.00の場合、Place要素は「存在しない」のと等しい。しかしマニフェストではVIEWTIFULの名前自体が⑤場所を最上位に置く哲学だと宣言している。

→ **矛盾**: 哲学上「場所が最重要」なのに、数学上「場所=1.00（無影響）」がデフォルトになっている。
→ **修正案**: 基準値を0.95にし、全ての場所がデフォルトで「やや不利」になる設計に。場所に投資した店舗だけが1.0以上を獲得できる構造。これにより「場所を選ぶこと自体が意思決定」という哲学が数学に反映される。

### 🔍 指摘④: Companion係数「一人ディナー = 0.95」は哲学的に危険

「一人で食べる美味しさ」を5%減ずるのは、「おひとりさま文化」が成熟した日本市場でネガティブに受け取られるリスクがある。また、カウンターフレンチ（SAT自身が高く評価するAn Di等）の価値を毀損しかねない。

→ **修正案**: 一人ディナー = 1.00（ニュートラル）とし、「共食のプラス」のみを係数化する。哲学的にも「引き算ではなく足し算」の方がSVDの思想に合致する。

### 🔍 指摘⑤: ベンチマーク分析にDen（傳）が不在

Asia's Best 2025で#1のSayan(Bangkok)、#2のDen(Tokyo)を分析対象にしていない。特に**Den（長谷川在佑）** はイノベーティブ日本料理であり、SVDとの比較対象として最も適切。Odetteはフレンチ、LogyはクロスカルチャーだがDenは「日本人が日本で、日本の文脈を再定義している」成功例。

→ **追加案**: Den / Sayan / Narisawa（里山キュイジーヌ＝サステナビリティ軸でSVDと最も近い）をベンチマーク追加。

### 🔍 指摘⑥: S-Score（サステナビリティスコア）が方程式に未統合

第3章でS-Scoreが定義されているが、v2.0方程式（A = Core × Place × Companion）にどう組み込まれるかが不明。独立指標なのか、それとも将来的にA値に統合されるのか。

→ **修正案**: 「S-ScoreはA値と並列の独立指標とする。A値は"今日の美味しさ"、S-Scoreは"100年後も美味しいか"を測る」と明記。これはSATの「100年先のSAPPORO」哲学と直結する。
