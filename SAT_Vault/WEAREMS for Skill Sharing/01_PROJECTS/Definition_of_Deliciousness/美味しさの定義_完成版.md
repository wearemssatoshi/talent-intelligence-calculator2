# 第1章: 美味しさの方程式 v3.0

### 30秒サマリー

> 美味しさとは、料理とサービスの相互作用（Core）が、場所のアイデンティティ（Place）と人との時間（Companion）によって増幅された体験的価値である。そのいずれかがゼロであれば、美味しさもまたゼロである。

### 方程式

```
A（美味しさスコア 最大550pt）= Core × Place × Companion

  Core（体験品質 最大400pt）= (FD① + FD②) × (SV③ + SV④)
    ├── FD① 料理の味わい（10pt）…五味の調和 / 風味 / 食感 / 供出温度 / 火入れの精度
    ├── FD② 料理の設え（10pt）…盛り付け / 食材の質 / 独自性
    │     └── BV（ビバレッジ）はFD②を補完する外部評価軸。
    │         Core計算には含めず、ルーブリックで独立採点し改善指標として運用する。
    ├── SV③ 空間づくり（10pt）…雰囲気 / 照明 / BGM
    └── SV④ オペレーション（10pt）…技術・知識 / タイミング / サービス力 / 組織力 / Welcome&Farewell / Recovery

  Place（場所係数 ×0.80〜1.25）
    └── SVDのアイデンティティを反映する係数

  Companion（共食係数 ×1.00〜1.10）
    └── LTCシーン別の体験増幅。共食のプラスのみを係数化

  ───────────
  理論値レンジ:
    Core最小(デフォルト6): (6+6)×(6+6) = 144
    Core最大: (10+10)×(10+10) = 400
    A最大: 400 × 1.25 × 1.10 = 550pt
  ───────────

  各カテゴリ（FD①②/SV③④）は各サブ項目の算術平均とし、10pt満点に正規化する。
```

### 満足度スコア C

```
C（満足度スコア）= A × B

  A = 美味しさスコア（上記）
  B = 期待ギャップ（0.5〜1.5）

  B > 1.0: 期待を超えた → 満足度がA値を上回る
  B = 1.0: 期待通り
  B < 1.0: 期待を下回った → 満足度がA値を下回る
```

# 第2章: 期待ギャップ B（0.5〜1.5）

```
B = f(料理期待ギャップ, Place期待ギャップ, ブランド期待ギャップ)
```

| | 定義 |
|---|------|
| B > 1.0 | 期待を上回った。A値以上の満足をゲストに提供 |
| B = 1.0 | 期待通り。A値がそのまま満足度 |
| B < 1.0 | 期待を下回った。A値は高くても不満が残る |

### B値の本質 — 顧客の認知であり、店の変数ではない（v3.1）

> **「期待値は『利用』するものであって、コントロールするものではない。」** — SAT

B値は**顧客の認知の中に存在する**ものであり、店側が直接操作できる変数ではない。名店であるほど価格とブランド力が期待を押し上げ、B > 1.0 の達成は構造的に困難になる。これは方程式の欠陥ではなく、現実の正確な反映である。

#### 定数と変数の分類

| 要素 | 性質 | 所有者 | 店にできること |
|------|------|--------|---------------|
| FD①② | 操作可能変数 | 店 | 日々の仕込みで磨く |
| SV③④ | 操作可能変数 | 店 | トレーニングで磨く |
| Place | 定数 | 立地 | 選択はできるが日々変えられない |
| Companion | 定数 | ゲストの利用目的 | コントロール権なし |
| **B** | **観察変数** | **顧客** | **操作不可。観察→利用→調整のみ** |

#### B値の扱い方 — 観察→利用→調整サイクル

店がB値に対して取るべきアプローチは「管理」ではなく、以下のサイクルである:

1. **観察する** — 顧客の期待がどこにあるかを客観的に理解する
2. **利用する** — 期待の位置を自分たちの行動の参照点にする
3. **調整する** — フレームワーク × 経験 × 実験で行動を調整し、顧客の反応を観察する

マーケットイン（顧客の期待を観察）とプロダクトアウト（自分たちの基準でA値を磨く）の両輪を回すことが、B値と向き合う正しい姿勢である。

**原則**: 店が磨くのはA値である。B値はA値の結果として、顧客の中に自然に生まれる。

### B値 測定プロトコル

| ステップ | 方法 | タイミング |
|---------|------|----------|
| ① 事前期待の把握 | 予約時の自然な会話からプロファイリング（シーン・目的・過去の来店歴）| 予約確定時 |
| ② 事後体験の記録 | 退店時の3問アンケート: 料理/場所/ブランドそれぞれの「期待との差」を5段階評価 | 退店時 or 翌日フォロー |
| ③ B値算出 | 3項目の平均を0.5〜1.5にスケーリング（5段階中3=1.0、5=1.5、1=0.5） | 月次集計 |
| ④ トレンド分析 | 月次B値の推移を監視。B値の下落はA値の成長がブランド認知に追い付いていないサイン | 四半期レビュー |

# 第3章: 並列指標 — S-Score & P-Score

### A値・S-Score・P-Scoreの三位一体

> **3指標が揃って初めて、SVDの哲学が実現される。**

```
A値 ── 今日の美味しさ（ゲスト体験）
S-Score ── 100年後も美味しいか（持続可能性）
P-Score ── スタッフは誇りを持てているか（人間の尊厳）
```

### S-Score（サステナビリティスコア）

> A値は「今日の美味しさ」を測る。S-Scoreは「100年後も美味しいか」を測る。

```
S-Score = 海洋持続性 + 大地循環性 + 文化継承性

海洋持続性: 北海道産海洋食材使用率、MSC/ASC認証食材比率
大地循環性: 地産地消率、フードロス率、エネルギー効率
文化継承性: 北海道食文化の継承活動、次世代育成プログラム
```

SATの原体験（釧路での昆布採り）に根ざした、SVD独自のサステナビリティ指標。Asia's Best 50の**Green Star賞**に直結。

### P-Score（Pride Score）— スタッフの誇り指標

> **「今の仕事を、家族に誇りを持って語れますか？」**

A値が高くてもP-Scoreが低ければ、SATの哲学「レストランで働く父を子が誇らしく思う」に反する。

```
P-Score = 定期サーベイ（四半期ごと）
  ├── 「今の仕事に誇りを感じていますか」
  ├── 「家族にこの仕事を語りたいですか」
  └── 「SVDで働いていることを嬉しく思いますか」
```
