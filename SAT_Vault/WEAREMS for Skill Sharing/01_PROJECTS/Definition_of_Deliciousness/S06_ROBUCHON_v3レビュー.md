## 5. 残存する課題（新規指摘含む）

### 🔍 新規指摘①: BVビバレッジ軸の方程式未統合

**問題**: ルーブリック v2.0にBV（ビバレッジ）が独立セクションとして追加されたが、方程式 Core = (FD① + FD②) × (SV③ + SV④) にBVがどう組み込まれるかが記載されていない。

- BVはFD②（料理の設え）に含まれるのか？ → 設えの定義に「ビバレッジ」は含まれていない
- SV④（オペレーション）の一部か？ → ソムリエの技術はSV④だが、ワインリスト自体はFD寄り
- 独立軸か？ → であれば方程式の構造変更が必要

**影響度**: 🟡 中。ルーブリックで評価はできるがA値に反映されない「孤児指標」になっている。

**修正案**: 以下のいずれかを明記
- (a) BVはFD②に含めて評価する（→ FD②のサブ項目を5つに拡張）
- (b) BVはA値とは独立した参考指標とする（S-Scoreと同様の位置づけ）
- (c) BVはSV④の「技術・知識」サブ項目でソムリエ力として評価する

### 🔍 新規指摘②: FD①のサブ項目列挙の不一致

**問題**: 方程式のFD①説明には「五味の調和 / 風味 / 食感 / 供出温度 / 火入れ」の5項目が列挙されているが、ルーブリック v2.0のFD①は「5+1サブ項目」として以下の6項目を定義:

```
1. 五味の調和
2. 風味（香り）
3. 食感
4. 供出温度
5. 火入れの精度 ✦ NEW
6. 技術的完成度    ← 方程式側の列挙に欠落
```

方程式の「(10pt)」と正規化ルール「算術平均→10pt満点」があるため計算上は問題ないが、列挙の不一致は読者の混乱を招く。

**修正案**: 方程式側のFD①説明を「五味の調和 / 風味 / 食感 / 供出温度 / 火入れ / 技術的完成度」の6項目に更新。
