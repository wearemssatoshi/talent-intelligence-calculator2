# 📖 Obsidian活用ガイド for SAT

---

## 🎯 基本の使い方

### 1. とりあえず書く
- **00_INBOX** にメモを放り込む
- 整理は後でOK

### 2. リンクで繋げる
- `[[ノート名]]` と書くだけでリンクになる
- 例：`[[SVD MINDFUL]]`

### 3. 整理する
- 関連するノートをフォルダに移動
- 完了したら `04_ARCHIVE` へ

---

## 📁 フォルダの使い分け

| フォルダ | 何を入れる？ | 例 |
|---------|------------|---|
| **00_INBOX** | とりあえずメモ | 思いつき、打ち合わせメモ |
| **01_PROJECTS** | 進行中の仕事 | SVD戦略、note記事の下書き |
| **02_AREAS** | 継続して管理するもの | 健康、学習、財務 |
| **03_RESOURCES** | 参考資料 | 読んだ本、記事、引用 |
| **04_ARCHIVE** | 完了したもの | 公開済みnote記事 |

---

## 🔥 SATにおすすめの活用法

### 1. note記事の執筆
```
00_INBOX → アイデアメモ
01_PROJECTS → 下書き・推敲
04_ARCHIVE → 公開後に保存
```

### 2. 打ち合わせメモ
- 毎日のデイリーノート（2026-02-04.md など）
- 後で `[[プロジェクト名]]` でリンク

### 3. SVDナレッジベース
- 店舗情報、スタッフ教育、メニュー開発
- すべてリンクで繋がる

### 4. Gとの共同作業
- Obsidianに書いておけば、Gが読んでフィードバックできる
- 記事の推敲、アイデアのブレスト、計画の整理

---

## ⌨️ よく使うショートカット

| 操作 | ショートカット |
|------|--------------|
| 新規ノート | Cmd + N |
| クイックオープン | Cmd + O |
| グラフビュー | Cmd + G |
| 保存 | Cmd + S |
| ファイルエクスプローラー | Cmd + Shift + E |

---

## 💡 コツ

1. **完璧を目指さない** - まず書く、整理は後で
2. **リンクを怖がらない** - 存在しないノートにリンクしてもOK（後で作れる）
3. **毎日少しずつ** - 習慣が大事
