# 🧭 SAT Vault

**伊賀智史（SAT）の知識ベース**

---

## 📁 フォルダ構造

| フォルダ | 用途 |
|---------|------|
| `00_INBOX` | とりあえずメモを放り込む場所。後で整理する |
| `01_PROJECTS` | 進行中のプロジェクト（SVD、MINDFUL、TSSなど） |
| `02_AREAS` | 継続的に管理する領域（健康、財務、学習など） |
| `03_RESOURCES` | 参考資料・蓄積した知識 |
| `04_ARCHIVE` | 完了したプロジェクト |
| `Templates` | ノートのテンプレート |

---

## 🔗 Gとの連携

このVaultは `/dotfiles/SAT_Vault` に配置されているため、**G（Antigravity）が直接アクセス可能**です。

- セッションログの保存
- ナレッジの参照・更新
- プロジェクト進捗の追跡

---

## 🚀 始め方

1. **INBOX に思いついたことを書く** - 整理は後でOK
2. **[[双方向リンク]]** を使ってノート同士を繋げる
3. **グラフビュー** で知識のネットワークを視覚化
