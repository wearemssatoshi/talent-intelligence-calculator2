## 📘 用語集（Glossary）

| 用語 | 定義 |
|------|------|
| **A値** | 美味しさスコア。A = Core × Place × Companion。理論最大値550pt |
| **Core** | 体験品質。(FD① + FD②) × (SV③ + SV④)。最大400pt |
| **FD** | Food Design（料理設計）。FD① = 味わい、FD② = 設え |
| **SV** | Service Value（サービス価値）。SV③ = 空間、SV④ = オペレーション |
| **Place** | 場所係数（×0.80〜1.25）。SVDのアイデンティティを反映 |
| **Companion** | 共食係数（×1.00〜1.10）。LTCシーン別の体験増幅 |
| **B** | 期待ギャップ。A値の成長速度 vs ブランド認知の成長速度 |
| **C** | 満足度スコア。C = A × B |
| **S-Score** | サステナビリティスコア。A値とは独立した並列指標 |
| **P-Score** | スタッフの誇り指標。「今の仕事を家族に誇りを持って語れるか」 |
| **ルーブリック** | 採点基準表。各サブ項目の10〜6段階の判定基準 |
| **北極星** | SVDの哲学：「北海道の海と大地が育む美味しさを、100年先のSAPPOROへ届ける」 |
| **LTC** | Life Time Customer。顧客生涯価値。シーン別Companion係数と連動 |
| **4レイヤー** | レストランの4分類：ガストロノミック / ビストロノミック / ブラッスリー / ケータリング |
| **Peak Side** | 藻岩山・大倉山エリアの事業群 |
| **Urban Side** | テレビ塔・赤れんがテラスエリアの事業群 |
| **NP** | Nuit Polaire（ヌーベルプース大倉山） |
| **JW** | The Jewels |
| **BQ** | BQ（赤れんがテラス） |
| **GA** | The Garden Sapporo |
