## 算出ロジック（2層統合モデル）

```
MP Point = ( KF① + KF② + KF③ ) / 3
```

| 要素 | 内容 |
|------|------|
| **KF①** 定数 | (季節指数 + 曜日指数 + 来場者指数) / 3 |
| **KF②** 売上 | 拠点内月別売上平均を 1.00〜5.00 正規化 |
| **KF③** 客数 | 拠点内月別客数平均を 1.00〜5.00 正規化 |
| **テレビ塔** | 天井方式: GA単体12月ピーク=5.00キャップ |
