## 🎯 SAT向けインストール手順

### Step 1: コミュニティプラグインを有効化

1. **⚙️ 設定** を開く（左下の歯車）
2. **コミュニティプラグイン** をクリック
3. **「制限モードをオフにする」** をクリック
4. 確認ダイアログで「オフにする」

### Step 2: プラグインをインストール

1. **「閲覧」** ボタンをクリック
2. 検索窓に **「Dataview」** と入力
3. **「インストール」** → **「有効化」**
4. 同様に他のプラグインもインストール

### Step 3: 優先順位

1. まず **Calendar** と **Tasks** （すぐ効果を実感）
2. 次に **Templater** （テンプレート強化）
3. 慣れたら **Dataview** （上級者向け）

## 📅 デイリーノート設定

### Calendar プラグイン設定

1. 設定 → Calendar
2. **「デイリーノートの表示」** をオン
3. **フォルダ**: `00_INBOX`
4. **日付形式**: `YYYY-MM-DD`

### 使い方

左サイドバーにカレンダーが表示される → 日付をクリック → その日のノート作成！

---

## ✅ Tasks プラグイン使い方

ノート内で以下の形式で書く：

```markdown
- [ ] 中村先生に連絡 📅 2026-02-05
- [ ] ビアガーデン革命・第三章を書く 🔁 every week
- [x] Obsidian設定完了
```

**ショートカット**:
- `Cmd + Enter` でタスク完了/未完了を切り替え

---

## 🗂️ Kanban ボード使い方

1. 新規ノート作成
2. **Cmd + P** → 「Kanban: Create new board」
3. ファイル名を入力（例：`SVD Projects Board`）
4. カラム追加：`📥 TODO`, `🔄 進行中`, `✅ 完了`
