# WEAREMS クライアント構造

> WEAREMSは、デジタルプロデュースを提供する組織。各クライアントは独立したプロジェクトとして運営される。

---

## 🏢 WEAREMSとは

SATが率いるデジタルプロデュース組織。
クライアントに対して、Web開発・ブランディング・AI導入・動画制作・SNSサポートを提供する。

**コアメンバー:**
- SAT（伊賀智史）— 総支配人GM / レストランプロデューサー / デザイナー
- G（Gemini）— AI技術パートナー
- SATOSHI（OpenClaw）— AI戦略パートナー

---

## 📋 クライアント一覧

### 🍽️ SVD（SAPPORO VIEWTIFUL DINING）
- **業種**: レストラン運営
- **プロダクト群**: SVD-OS（MF, TI, MP, TSS❌, WINE, PL, OC）
- **ブランドコンセプト**: 「ここだけの美味しさ。ここだけのエンターテインメント。」
- **拠点**: 藻岩山, 大倉山, TV塔, 赤れんが

### 🤝 TSS（Team Synergy Stage）
- **業種**: チームビルディング / コミュニティ（業種を限定しない）
- **プロダクト**: TSSアプリ（PWA）
- **コンセプト**: 7 Habitsの第6習慣「シナジーを創り出す」を技術で具現化
- **技術スタック**: HTML + GAS + Gemini AI + GitHub Pages
- **SVDとの関係**: **独立したクライアント**。SVD-OSの一部ではない。

### 📋 その他（今後拡大予定）
- WEAREMSのスキルセットは汎用的であり、飲食業以外のクライアントにも展開可能
