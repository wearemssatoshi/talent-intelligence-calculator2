①

ROSTER — チーム能力可視化

チーム一覧

チーム全員の能力をCARDビューとSKILL MAPビューの2つの視点で一望できます。

  • CARDビュー: 各スタッフの名前、役職、Total CP、P/S/E/Mレーダーチャート
  • SKILL MAPビュー: 全員×24スキルのヒートマップ。チーム全体の強み・弱みが一目瞭然
  • フィルタ: 拠点別（JW, NP, BQ, GA…）、ロール別で絞り込み
  • ソート: Total CP順、カテゴリ平均順で並べ替え
🎯 使い方: まず拠点を選び、チームの全体像を把握。SKILL MAPで弱いカテゴリを特定し、面談（②ASSESS）へ進みます。
②

ASSESS — 自己評価リビュー

面談ツール

面談時に、過去の評価を振り返りながら今後のアプローチを確認するためのフォーマットです。

⚠️ 注意: ここでは評価の「変更・確定」はしません。評価の入力はCalculatorで行います。
  • Self / MGR 切り替え: 左パネルのボタンで、自己評価とMGR評価を瞬時に切り替え
  • 読み取り専用バー: 各スキルのスコアをゴールドのプログレスバーで視覚表示
  • LV1〜10 目安（アコーディオン）: ▶ をクリックすると、各LVの達成基準が展開。現在のLVはゴールドでハイライト
  • ルブリック: 各スキルの評価観点を常に表示。面談の客観的な根拠に
  • 面談メモ: 面談の記録を残せるテキストエリア
🎯 面談の流れ: スタッフ選択 → Self評価を確認 → MGRに切替えて比較 → LV目安でギャップの根拠を確認 → 面談メモに記録
③

GROWTH — 成長GAP可視化

成長比較

2つの評価時点を比較し、成長GAP（どれだけ伸びたか）と評価GAP（SelfとMGRのズレ）を可視化します。

  • 成長GAP: 3ヶ月前と最新のMGR評価を比較 → どのスキルが伸びたか
  • 評価GAP: 最新のSelfとMGRを比較 → 自己認識とMGR認識のズレを発見
  • 重ねレーダーチャート: From（青）とTo（金）を重ねて視覚比較
  • Diff Table: 24スキル全ての詳細スコアとGAP値を一覧表示
  • GAP表示: プラスは🟢、マイナスは🔴で色分け。小数点第2位まで表示

SETTINGS — GAS接続設定

設定

GASバックエンドURLを設定し、接続を確認します。

  • GAS URL入力: バックエンドのデプロイURLをペースト
  • TEST接続: ボタン1つでGAS接続を確認。成功すると緑バッジに変化
  • ローカル保存: URLはブラウザのlocalStorageに保存。毎回入力不要
💡 初回セットアップ: まず⑤SETTINGSでGAS URLを設定 → TESTで接続確認 → ①ROSTERに移動してチームデータを読み込み
