# Wine List Generator

納品書（PDF/画像）からワインリストを自動作成するWebアプリケーションです。

## 機能

1. **納品書アップロード**: PDFまたは画像（JPG/PNG）をドラッグ＆ドロップで読み込み。
2. **OCR自動認識**: Tesseract.jsを使用し、ワイン名、ヴィンテージ、価格などを自動抽出。
3. **データ編集**: 抽出されたデータを表形式で確認・修正可能。
4. **CSVエクスポート**: Excelで開けるBOM付きCSVを出力。
5. **PDFワインリスト作成**: ソムリエ向けの美しいワインリスト（A4縦）を自動生成。

## 技術スタック

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **OCR**: Tesseract.js
- **PDF**: @react-pdf/renderer

## セットアップ手順

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ブラウザでアクセス:
   http://localhost:3000

## 使い方

1. 画面中央のエリアに納品書の画像またはPDFをドラッグ＆ドロップします。
2. OCR処理が始まり、読み取り状況がバーで表示されます。
3. 処理が完了すると、抽出されたデータが下のテーブルに表示されます。
4. 必要に応じてデータを修正・追加・削除してください。
5. 右上の「原価率設定」でマークアップ率を変更できます（デフォルト1.8倍）。
6. 「CSVエクスポート」でデータを保存するか、「ワインリストPDF作成」でメニューを生成します。

## 注意事項

- OCRの精度は画像の画質に依存します。手書き文字や複雑なレイアウトの場合、正しく読み取れないことがあります。
- PDF生成時のフォントはGoogle Fonts（Noto Sans JP, Noto Serif）を使用しています。インターネット接続が必要です。
